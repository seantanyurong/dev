# Overview

This is a collection of scripts I use to set up and maintain my development environment.

## dev-env

This script is used to set up a new laptop with the specific configs like `wezterm`, `nmvim`, and `zshrc` in my dev environment `/env`

## run

This script is used to run all the scripts in the `/runs` directory, mainly for downloading packages like `nvim` for a fresh setup.

## misc

This directory contains miscellaneous configs.

## guide to setup a new device

1. install arc
2. install brew via terminal
3. install notion
4. install moom
5. install raycast - port over config
6. install telegram
7. install whatsapp
8. install devtools from terminal
9. clone this repo
10. run ./run
11. run ./dev-env
12. accept all the permissions for karabiner, etc.
